This is the core code for displaying messages in a `GroupServer`_
group. This module supplies:

* :ref:`A viewlet manager <viewlet>` for displaying the *Messages*
  tab on the *Group* page, and
* :ref:`Some utilities <utilities>` that are useful for
  displaying messages.

:doc:`temporal` is also discussed here, because it mostly comes
up with the design of the pages associated with messages.
